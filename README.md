🌦️ Kolkata Climate Report

Overview
A Power BI dashboard that provides insights into Kolkata's climate using historical weather data. The report visualizes temperature, rainfall, humidity, and other weather trends to support climate analysis.

Features
- Temperature trend analysis
- Rainfall and humidity visualization
- Monthly and yearly climate insights
- Interactive filters and charts
- Easy-to-understand dashboard

 Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

Project Files
- `Kolkata Climate Report.pbix` – Power BI dashboard file

How to Use
1. Download the `.pbix` file.
2. Open it using Microsoft Power BI Desktop.
3. Explore the dashboard using the interactive visuals and filters.
